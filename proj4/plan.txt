An FCB corresponds to each file. It contains
    A an id (index), file size, and file name
    A pointer to an index block
    
An index block contains 4 byte ints full of pointers to data blocks.

How to create a file:
    Create a direntry
    Find free block 
    Turn free block into an index file
    Find free inode (using inode map)
        Choose ID from max ID + 1
        Assign name
        Assign pointer to index block
    

How to do a write:
    Write basics, edit and expand the file so that these new n bytes are also placed in the file
    In memory you could just do expand array
    Make sure that you have an index block per file before
    First if you dont have space expand
        So if len < data avilable (do these calculations from total len)
        Create a new data file and add it to index
    Then write to the last avilable data file as much as you can
    If you run out of space expand again (repeat)

    Update the space and make sure you write back
